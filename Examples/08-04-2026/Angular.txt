# Data Binding : 
    - connect UI to code 
    - There are 2 types of data Binding
        1. One way 
            - interpolation
                Show data from ts in html
                {{value}}
                
            - Property Binding
                Bind HTML Property to TS variable
                [Property]=value
            - Event Binding
                call function on event
                (click)=()
        2. Two way 
            - ng module
                Sync data both ways - Ts to html and html to ts
                [(ngModel)]="value"
